# Hotel-Booking-System
# Hotel Booking System has Following Features

 - Book Hotel Online
 - Online Payament Method
 - Check Booking Status
 - View Hotel Rooms 
 - Check Location and Other Detais
 
 # Frame Work 
 - Back Hand Python Django
 - Font Hand Html Css,Javascrip and jQuery
 
 # Database
 - Sqlite3
